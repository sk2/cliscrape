    - `tests/fixtures/textfsm/explicit_eof_rules.textfsm` demonstrating EOF rules run once.

  </action>
  <verify>`cargo test -q` passes and both EOF fixtures are exercised by tests.</verify>
  <done>EOF behavior matches TextFSM: explicit empty EOF suppresses implicit record; explicit EOF rules execute once.</done>
</task>

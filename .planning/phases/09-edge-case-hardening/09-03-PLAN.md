# Plan 09-03: Threshold Validation and Timeouts

Goal: Implement the logic for field coverage thresholds and timeouts.

## Research
- `Template::parse_internal` in `src/engine/fsm.rs` is the main parsing loop.
- `calculate_coverage` already exists in `tests/coverage.rs`.

## Strategy
1. Integrate coverage calculation at the end of `parse_internal`.
2. Compare coverage to `threshold` and emit warning or error (in `strict` mode).
3. Implement a timeout check inside the `while line_idx < lines.len()` loop.
4. If a timeout occurs, return `ScraperError::Timeout`.

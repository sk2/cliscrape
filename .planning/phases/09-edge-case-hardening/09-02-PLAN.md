# Plan 09-02: CLI Integration for Hardening

Goal: Add CLI flags for `strict` mode, `threshold`, and `timeout`.

## Research
- `src/cli.rs` defines the CLI arguments.
- `src/main.rs` implements the commands.

## Strategy
1. Add `strict` (boolean) to the `Parse` command in `src/cli.rs`.
2. Add `threshold` (f64, default 80.0) to the `Parse` command in `src/cli.rs`.
3. Add `timeout` (u64, milliseconds) to the `Parse` command in `src/cli.rs`.
4. Update `main.rs` to parse and pass these flags to the parser.

## Tasks
1. [ ] Update `src/cli.rs` with new flags.
2. [ ] Update `src/main.rs` to handle these flags.
3. [ ] Implement a `ParseOptions` struct to pass these to the `FsmParser`.

## Verification
- `cliscrape parse --help` shows new flags.
- `cliscrape parse --strict` works (parsed value not yet used).
- `cliscrape parse --threshold 50.0` works.
- `cliscrape parse --timeout 1000` works.

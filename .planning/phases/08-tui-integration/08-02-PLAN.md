# Plan 08-02: Metadata and Preview in Template Browser

Goal: Metadata extraction and display in the template browser.

## Research
- `cliscrape::template::metadata::extract_metadata` provides metadata for a template.
- `cliscrape::template::library::get_embedded` provides content for embedded templates.

## Strategy
1. Add `TemplateMetadata` to `TemplateBrowserEntry`.
2. Extract metadata for all templates on browser initialization (or on demand for performance if list is large).
3. Display metadata (description, compatibility, version) in a details pane in the browser UI.
4. Display a preview of the template content.

## Tasks
1. [ ] Update `TemplateBrowserEntry` in `src/tui/browser.rs` to include metadata and optional preview content.
2. [ ] Extract metadata for templates in `TemplateBrowserState::new`.
3. [ ] Update `ui::draw_template_browser` in `src/tui/ui.rs` to include metadata and preview panes.

## Verification
- Browser UI shows metadata (description, etc.) for the selected template.
- Browser UI shows a preview of the template content.
- Metadata and preview update as the user navigates the list.

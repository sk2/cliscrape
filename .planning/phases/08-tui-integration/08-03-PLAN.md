# Plan 08-03: XDG User Directory Support in Template Browser

Goal: Load and browse templates from the XDG user directory.

## Research
- `cliscrape::template::resolver::TemplateResolver` provides XDG directory discovery.
- `xdg::BaseDirectories` used in `TemplateResolver`.

## Strategy
1. Extend `TemplateBrowserState` to list templates from the XDG user directory.
2. Mark entries as "User" or "Embedded".
3. Use `TemplateResolver` logic to find templates in the user data directory.

## Tasks
1. [ ] Update `TemplateBrowserState` to list files from `~/.local/share/cliscrape/templates/`.
2. [ ] Identify file format (YAML/TOML/TextFSM) from file extension.
3. [ ] Extract metadata and preview content for user templates.
4. [ ] Merge user templates into the browser list (with visual indicator).

## Verification
- User-placed templates in `~/.local/share/cliscrape/templates/` appear in the browser.
- Metadata for user templates is correctly extracted and displayed.
- User templates have a different label or color in the list.

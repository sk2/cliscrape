# Plan 08-01: Template Browser Foundation

Goal: Implement `TemplateBrowserState` and basic UI for listing embedded templates.

## Research
- `cliscrape::template::library::list_embedded` provides the list of embedded templates.
- `src/tui/picker.rs` provides a template for list-based UI states.

## Strategy
1. Define `TemplateBrowserState` in a new file `src/tui/browser.rs`.
2. Populate the state with embedded template names initially.
3. Update `AppState` to include `TemplateBrowserState`.
4. Add `TemplateBrowser` mode to `Mode` enum in `src/tui/app.rs`.
5. Implement basic drawing for `TemplateBrowser` in `src/tui/ui.rs`.
6. Add key handling for navigating the browser in `src/tui/mod.rs`.

## Tasks
1. [ ] Create `src/tui/browser.rs` with `TemplateBrowserState`.
2. [ ] Update `src/tui/app.rs` with `TemplateBrowserState` and `Mode::TemplateBrowser`.
3. [ ] Implement `ui::draw_template_browser` in `src/tui/ui.rs`.
4. [ ] Handle `Mode::TemplateBrowser` in `handle_key` in `src/tui/mod.rs`.
5. [ ] Add hotkey (e.g., `t`) to open template browser from `Browse` mode.

## Verification
- Pressing `t` in `Browse` mode opens the template browser.
- Browser lists all embedded templates.
- Up/Down keys navigate the list.
- `Esc` or `q` exits the browser back to `Browse` mode.
